# Safety and Ethics Plan

## Project Safety Rules

This project will follow safety-first engineering principles.

1. The drone will first be tested in simulation before real flight.
2. Real-world testing will begin with manual flight only.
3. Autonomous testing will happen at low speed and low altitude.
4. A manual override or emergency stop must always be available.
5. Propeller guards will be used whenever possible.
6. Testing will happen in a controlled environment away from people, animals, vehicles, and fragile objects.
7. Outdoor testing will follow all relevant drone rules and school/local restrictions.

## Ethics Rules

This project will not be designed as a privacy-invasive surveillance system.

The main demonstration target will be a non-human object, such as:

- colored box
- printed marker
- toy car
- backpack
- AprilTag
- ArUco marker

## Human Privacy

Face verification or human identification will not be used as the main autonomous tracking demonstration. If human-related testing is ever discussed, it must involve consent, privacy protection, and science fair/school approval.

## Responsible Project Framing

This project should be described as:

“Autonomous object tracking and navigation using AI computer vision.”

It should not be described as:

“AI surveillance drone for tracking people.”

## Risk Reduction

The project will prioritize:

- simulation first
- low-speed testing
- controlled environments
- non-human targets
- clear documentation
- safe shutdown procedures
