# Project Proposal

## Title

AI Autonomous Drone for Real-Time Object Tracking and 3D-Aware Navigation

## Problem Statement

Autonomous drones are becoming increasingly important in robotics, search-and-rescue, inspection, delivery, environmental monitoring, and emergency response. However, many advanced autonomous drone systems require expensive hardware, complex software, and professional-level engineering resources. This project investigates whether a low-cost drone system can use artificial intelligence, computer vision, and sensor-based navigation to identify and track a selected object in real time.

## Purpose

The purpose of this project is to design, build, and test an AI-assisted drone system that can recognize a selected non-human target, follow its movement, and navigate safely in a controlled environment. The project will combine robotics, machine learning, computer vision, aerodynamics, embedded systems, and path planning.

## Research Question

How effectively can a low-cost autonomous drone use real-time computer vision and sensor-based navigation to track a selected object while maintaining stable and safe movement in a controlled environment?

## Hypothesis

If a drone is equipped with a camera, onboard computing, and an object tracking model, then it should be able to identify and follow a selected target in real time with measurable accuracy, as long as the environment is controlled and the target is clearly visible.

## Main Goals

1. Research drone aerodynamics, flight control, and sensor systems.
2. Select or build a safe drone platform for experimentation.
3. Implement real-time object detection and tracking.
4. Test autonomous navigation in simulation before real-world flight.
5. Add basic path planning and obstacle avoidance.
6. Collect performance data such as tracking accuracy, response time, distance error, and failure rate.
7. Write a research paper based on the system design, experiments, and results.

## Scope

This project will focus on controlled object tracking rather than unrestricted surveillance. The target will be a non-human object such as a box, marker, toy vehicle, or printed tag. The system may later be extended to more advanced perception tasks, but the main science fair demonstration will avoid privacy-invasive human tracking.

## Expected Outcome

By the end of the summer, the expected outcome is a working prototype or simulation-supported system that demonstrates real-time object tracking and basic autonomous drone navigation. The final deliverables will include code, GitHub documentation, experiment results, videos, diagrams, and a research paper draft.

## Project Deliverables

* GitHub repository
* Project roadmap
* Safety and ethics documentation
* Parts list and hardware notes
* Simulation environment
* Object tracking model
* Navigation control code
* Experiment results
* Demo video
* Research paper draft
